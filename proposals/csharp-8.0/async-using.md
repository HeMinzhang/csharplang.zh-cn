## <a name="async-using-declaration"></a>异步 using 声明

在C# 8.0 中，我们添加了对*异步 using*语句的支持。 有两种形式。 其中一个块体是使用 declaratation 的作用域。 另一个声明局部，并隐式限定为块的结尾。 这是其规范的占位符。
